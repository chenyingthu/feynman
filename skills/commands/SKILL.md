# Commands

Browse the complete command catalog.

## Activation

- When: User wants to see all available commands
- Input: None (opens interactive browser)
- Flags: None

## Workflow

Executes directly via UI:
1. Aggregate commands from all sources (built-in, packages, extensions)
2. Categorize by source and function
3. Present interactive browser with descriptions

## Output

- **Display**: Complete command catalog
- **UI**: Interactive command browser

## Notes

This is a utility command; no persistent output generated.
