# Help

Interactive command browser and help system.

## Activation

- When: User needs to discover available commands
- Input: None (opens interactive selector)
- Flags: None

## Workflow

Executes directly via UI:
1. List all available slash commands by category
2. Allow interactive selection
3. Prefill editor with selected command

## Output

- **Display**: Grouped command reference
- **UI**: Interactive command selector

## Notes

This is a utility command for command discovery; no persistent output generated.
