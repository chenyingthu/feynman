# Tools

Browse available tools and their specifications.

## Activation

- When: User wants to discover what tools are available
- Input: None (opens interactive browser)
- Flags: None

## Workflow

Executes directly via UI:
1. List all callable tools
2. Show source package for each tool
3. Display parameter summaries

## Output

- **Display**: Tool catalog with specifications
- **UI**: Interactive tool browser

## Notes

This is a utility command; no persistent output generated.
